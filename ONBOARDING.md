# Squad AI — Vijay Chauhan's SEO Agent Stack

You're talking to **Pinchy**, the orchestrator of a 24-agent SEO AI system built for Busbud.com.

---

## What this system does

Pinchy routes every request to the right specialist automatically. You don't need to know which agent to use — just describe what you need.

**Examples to try right now:**
- `Morning briefing` — surfaces top 3 priorities, ranking changes, pending tasks
- `What are our top quick wins this week?` — GSC data, pages ranking 11-20
- `Audit busbud.com/en-us technically` — crawl + E-E-A-T + NavBoost check
- `Write a content brief for "bus from Montreal to Toronto"` — full SEO brief
- `What does FlixBus rank for that we don't?` — competitor gap analysis
- `Check if our homepage is indexed correctly` — GSC URL inspection

## Key slash commands

```
/crawl-page [url]              Full SEO audit of any URL
/serp-scrape [keyword]         Scrape top 10 + PAA + rich features
/full-audit [url]              NavBoost + E-E-A-T + AI Overview audit
/eeat-check [url]              E-E-A-T score 0–100 with specific fixes
/content-gap [kw] | [url]      Topics competitors cover that you don't
/pipeline-status               Full system health: DBs, crons, errors
/dashboards                    Check/start all 3 live dashboards
```
