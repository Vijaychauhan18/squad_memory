# Squad Agents — Reference Guide

All 24 agents in Vijay's AI stack. Pinchy (the orchestrator) automatically routes your request to the right specialist.

## How to talk to agents
Just describe your task in plain English. Pinchy reads it and delegates.

Examples:
- "Check rankings for busbud.com this week" → John Mueller
- "Find quick win keywords we're ranking 11-20 for" → Quick Wins Hunter
- "Write a blog post about bus travel in Mexico" → Plankton
- "Audit the technical SEO on busbud.com/en-us" → Technical Auditor
- "What are our competitors ranking for that we're not?" → Competitor Spy
