## Summary
The Microsoft Live.com link command operator is currently offline and has been offline since yesterday morning. Kevin Gibbons reported yesterday that a link:https://www.msn.com returns a blank page. I tested more examples and have seen it down since at least 5:27 AM (EST) yesterday morning. All other searches seem to work, including a site command.
