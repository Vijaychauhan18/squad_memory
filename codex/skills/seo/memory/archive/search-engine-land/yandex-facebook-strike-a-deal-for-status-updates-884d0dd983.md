## Extracted Body
Yandex, the number one search engine in Russia, has struck a deal with Facebook to get some updates for its search engine index. As announced today, the deal has two parts:

First, Facebook will supply Yandex with a data feed of status updates from Facebook Pages. This is similar to the Facebook-Google agreement announced last December, but falls short of Facebook’s agreement with Bing, which also includes updates on personal Facebook profiles. Yandex says it will add the Facebook data into its blog search index.

Second, the Yandex.ru home page is showing a Facebook content widget (see image below) and Yandex’s instant messaging tool will offer Facebook notifications.

The deal only appears to involve Yandex’s Russian search engine at Yandex.ru, not its Yandex.com international version.
