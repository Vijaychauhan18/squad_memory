## Extracted Body
The Yahoo Search Blog announced that Yahoo Slurp , Yahoo’s web crawler, is moving from inktomisearch.com to crawl.yahoo.net.

In short, the user agent for Yahoo’s crawler will remain “Yahoo! Slurp.” However, reverse DNS lookups on the IP address by the true Yahoo crawler will now resolve to crawl.yahoo.net. The transition should take “a few weeks.”
