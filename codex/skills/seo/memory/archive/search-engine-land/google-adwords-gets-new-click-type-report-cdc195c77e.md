If you take advantage of some of the relatively new AdWords features like click-to-call, Sitelinks, and Product Ads, there’s a new report in your AdWords account to show how each one is performing. It’s called “Click Type” and you can access it by going to Campaigns > Segment > Click Type . Here’s a screenshot from Google’s announcement :
