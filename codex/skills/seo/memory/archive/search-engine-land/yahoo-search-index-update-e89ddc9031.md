The Yahoo Search Blog officially confirmed the recent speculations of an index update taking place. Priyank Garg of Yahoo says, that Yahoo is “in the process of rolling out some changes to our search results. As usual, you may be seeing some changes in ranking as well as some shuffling of the pages that are included in the index throughout this process.” The update should be done “very soon” according to Priyank.
