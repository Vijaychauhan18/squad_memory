The Google App for Windows has been officially launched outside the restrictions of Google Labs. This AI-Mode-first search tool is not a browser and does not include Gemini but does add speech input, image input and screen-sharing input alongside typed input. This is the first search input surface from Google that defaults to AI-generated output.

It’s a pure search tool so Gemini is not currently included.

The app separates AI Mode search from the browser completely. It removes ad clutter, it’s clean. It also includes Google Lens and pushes the idea of image search via screen / app sharing. Voice search is integrated.

it it a browser? No. It keeps the user in search and only opens a browser when a link is clicked. Advantage Google. The data is generates for them must also be part of the reason it’s being rolled out.

One wonders whether this is the final test before AI Mode becomes default in other Google search surfaces and, as a result, shifts the data-set towards AI. At the very least it shows that Google has confidence in AI Mode.
