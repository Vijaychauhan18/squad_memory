## Extracted Body
Hobo SEO Tools is like a Swiss Army Knife of SEO Tools that operates entirely in Google Workspace (your Gmail account). It’s 100% private to you. Just set up your free Google API, add the project ID to your sheet, and you’re all set. Track and record all your work in one client report that you continually update. Any tools I make in future will go in this tool, and early-birds get lifetime tool updates .

Hobo SEO Tools V1 is a toolset and an SEO Audit framework in a Google Sheet that helps you audit any website and publish client reports.

This is a limited offer. There are no subscription fees. It is a one-off payment, and you can use it on multiple clients. V1 comes complete with the Hobo SEO Audit Checklist, the Hobo EEAT tool, and a selection of other SEO Tools, all working in Google Sheets, designed to work well within your Google API quota:

If you have a lot of clients and need autonomous, automatic reporting and client emailing, you will love its sister tool, the Hobo SEO Dashboard , which is an autonomous version of Hobo SEO Tools, (without the client emailing functions and some other automations in the Dashboard). Essentially, with Hobo SEO Tools, you press the buttons, and with Dashboard, it runs on its own, once set up. With both of these tools, you set up your free GSC API yourself for maximum privacy.

Not comfortable with APIs or want Hobo to handle it all for you? Check out Hobo SEO Auditor , which is a hosted (lite) version of the Hobo SEO Dashboard that Hobo runs for you. We just need access granted to your client’s SC data ( under strict NDA ).

Q: Why would I get the Hobo SEO dashboard rather than Hobo SEO Tools (£99 one-off) – I have anywhere from 4-10 clients, often they don’t need any reporting, but I can see the benefit of me seeing the reports. I don’t have an SF subscription.

A: Hobo SEO Dashboard is an autonomous, automatic reporter. Set it up, and it will continuously monitor and publish reports – and send them to a schedule you set – to your ten clients. As you work on audits, the system tracks and records progress. All the tools work with Screaming Frog if you have it. Hobo SEO Auditor, FYI, is a service I run for you if you don’t want the hassle of setting up any APIs. Reports are only sent to you. Hobo SEO Tools V1 doesn’t have autonomy . Hobo SEO Tools V1 IS the tool set, where Hobo SEO Dashboard V4 is a robot that USES the tool set you set up at your end, and Hobo SEO Auditor (an instance of Dashboard in my account) uses the tool set I set up at my end . So in short, if you want automatic reporting, you go with either SEO Dashboard (one-off payment) or SEO Auditor (monthly subscription) . If you don’t need automation at all, then Hobo SEO Tools V1 lets you produce the reports yourself in minutes. Hobo SEO Tools V1 is a great tool to install in another Gmail account (for instance) and use that account if you want to go deeper with any particular project (which leaves the Dashboard to run uninterrupted). You can also use both in the same account, though it will probably interfere with API quotes and result in temporary pausing of the Dashboard and SEO Tools functionality until the API resets for the day. Hobo SEO Dashboard does a lot of complex automation that is not available in Hobo SEO Tools V1. All are built on Hobo SEO Dashboard technology.

Q: Can I use Hobo SEO Tools V1 on multiple Gmail Accounts? A: Yes, but you need to make a copy of Hobo SEO Tools (or Dashboard) for each account you have access to. This is a core privacy feature of Google Sheets.

Q. How much does Hobo SEO Tools V1 cost to run? A: Nothing. All Hobo SEO Tools and Dashboards are designed to work within your API daily quota as set by Google (every Gmail account has this, even free ones). They are designed to be extremely frugal with API limits.

Q. Does Hobo SEO Tools V1 run in a free Gmail account? A: Yes, though some of the more time-intensive tasks work better with a paid Google Workspace account.

Q. Can Hobo SEO Tools V1 do everything that Hobo SEO Dashboard can do? A: No. The main difference is that Hobo SEO Tools V1 is not an autonomous robot, whereas Hobo SEO Dashboard is. Hobo SEO Tools V1 is your tool to take 100% control of auditing and publishing. Hobo SEO Tools V1 uses the same technology as Hobo SEO Dashboard, and it does contain a lot of automation itself, although Hobo SEO Tools is not autonomous.

Q. Does Hobo SEO Tools V1 have the same tools as in Hobo SEO Dashboard? A: Yes. You can sync search console accounts, you can import Screaming Frog files, and you can publish reports.

Q. Does Hobo SEO Tools V1 use AI? A: No. There is no AI in any Hobo SEO Dashboard-powered product. It is 100% robotic (hand-coded). Why? Because AI just makes stuff up .

Q. Is Hobo SEO Tools V1 private? A: Yes. None of your data leaves your own Google account. Hobo does not track, collect, store or use your own private data for any purpose. Your privacy is one of the core reasons behind Hobo SEO Dashboard technology.
